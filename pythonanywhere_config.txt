# PythonAnywhere Configuration Guide

1. Web App Configuration:
   - Domain Name: saskatoongaragedoorexpets.ca
   - Python Version: 3.11
   - Framework: Flask
   - WSGI Configuration Path: /var/www/saskatoongaragedoorexpets_ca_wsgi.py

2. Static Files Configuration:
   URL: /static/
   Directory: /var/www/saskatoongaragedoorexpets.ca/static/

3. Environment Variables Setup:
   - FLASK_ENV=production
   - FLASK_SECRET_KEY=[your-secret-key]
   - DATABASE_URL=[your-database-url]
   - MAIL_USERNAME=[your-mail-username]
   - MAIL_PASSWORD=[your-mail-password]
   - PGHOST=[your-pg-host]
   - PGPORT=[your-pg-port]
   - PGUSER=[your-pg-user]
   - PGPASSWORD=[your-pg-password]
   - PGDATABASE=[your-pg-database]

4. Required Python Packages:
   - See requirements.txt for complete list

5. Deployment Steps:
   a. Upload project files to: /var/www/saskatoongaragedoorexpets.ca/
   b. Configure static files in Web tab:
      - Add mapping: /static/ -> /var/www/saskatoongaragedoorexpets.ca/static/
   c. Enable HTTPS in Web tab
   d. Set up environment variables in Web tab
   e. Install requirements: pip install -r requirements.txt
   f. Reload web application

6. Database Migration:
   - Run migrations using Flask-Migrate:
     $ flask db upgrade

7. SSL/HTTPS:
   - Enable HTTPS through PythonAnywhere web interface
   - HSTS is configured in wsgi.py
   - Force SSL redirect is implemented

8. File Permissions:
   - Ensure proper permissions for log directory:
     $ chmod 755 logs/
   - Set proper permissions for static files:
     $ chmod 644 static/*

9. Monitoring:
   - Log files location: /var/www/saskatoongaragedoorexpets.ca/logs/
   - Error log: /var/log/saskatoongaragedoorexpets.ca.error.log
